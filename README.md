# SCT_CS_1 - Caesar Cipher

## Task Objective
Create a Python program that encrypts and decrypts text using the Caesar Cipher algorithm.

## Features
- Encrypts text using a user-defined shift value.
- Decrypts encrypted text back to the original message.
- Supports uppercase and lowercase letters.
- Preserves spaces, numbers, and special characters.

## Technologies Used
- Python
- Google Colab
- GitHub

## Example

Input:
Message: Hello World
Shift: 3

Output:
Encrypted Message: Khoor Zruog
Decrypted Message: Hello World
